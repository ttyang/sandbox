Returnable

Description

Describes types that can be used as the return type of a function.

Note

A move-only type that can be constructed from an rvalue reference can be Returnable without being CopyConstructible.

Section in N2914

The corresponding definition of Returnable may be found in the N2914 section [concept.support].

Differences From N2914

In N2914 this concept is explicit with concept maps implicitly defined for the implementation's arithmetic types. In Boost.Generic, this concept is auto and is defined via the standard's type_traits facilities and True.
