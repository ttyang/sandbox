ObjectType

Description

Describes object types, for which storage can be allocated.

Definition

Table 1.12. ObjectType Definition

Boost.Generic Definition

N2914 Definition 

BOOST_GENERIC_AUTO_CONCEPT
( ( namespace boost, generic, std_concept )
, (ObjectType)( (typename) T ), ( public (ReferentType< T >) (PointeeType< T >) )
, ( requires True< std::is_object< T >::value > )
)


concept ObjectType<typename T> : ReferentType<T>, PointeeType<T> { }

// Requires: for every type T that is an object type, a concept map
// ObjectType<T> is implicitly defined in namespace std.


Header 
#include <boost/generic/std_concept/support_concepts/object_type.hpp>
Section in N2914

The corresponding definition of ObjectType may be found in the N2914 section [concept.support].

Differences From N2914

In N2914 this concept is explicit with concept maps implicitly defined for the implementation's arithmetic types. In Boost.Generic, this concept is auto and is defined via the standard's type_traits facilities and True.
