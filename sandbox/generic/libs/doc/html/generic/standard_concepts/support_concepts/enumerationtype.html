EnumerationType

Description

Describes enumeration types.

Definition

Table 1.5. EnumerationType Definition

Boost.Generic Definition

N2914 Definition 

BOOST_GENERIC_AUTO_CONCEPT
( ( namespace boost, generic, std_concept )
, (EnumerationType)( (typename) T )
, ( public IntegralConstantExpressionType< T > )
//, ( (IntegralType) underlying_type, int )
, ( (IntegralType) underlying_type, typename std::underlying_type< T >::type )
, ( requires True< std::is_enum< T >::value > )
)


concept EnumerationType<typename T> : IntegralConstantExpressionType<T> {
  IntegralType underlying_type;
}

// Requires: underlying_type is the underlying type of the enumeration type.
//
// Requires: for every type T that is an enumeration type, a concept map
// EnumerationType<T> is implicitly defined in namespace std.


Header 
#include <boost/generic/std_concept/support_concepts/enumeration_type.hpp>
Section in N2914

The corresponding definition of EnumerationType may be found in the N2914 section [concept.support].

Differences From N2914

In N2914 this concept is explicit with concept maps implicitly defined for the implementation's arithmetic types. In Boost.Generic, this concept is auto and is defined via the standard's type_traits facilities and True.
