ValueType

Description

Describes value types, for which objects, variables, references, pointers, and pointers-to-members can be created.

Definition

, (ValueType)( (typename) T )

concept ValueType<typename T> : ObjectType<T>, VariableType<T>, MemberPointeeType<T> { }

// Requires: for every type T that is an object type but not an abstract class,
// a concept map ValueType<T> is implicitly defined in namespace std.

Header 
#include <boost/generic/std_concept/support_concepts/value_type.hpp>
Section in N2914

The corresponding definition of ValueType may be found in the N2914 section [concept.support].

Differences From N2914

In N2914 this concept is explicit with concept maps implicitly defined for the implementation's arithmetic types. In Boost.Generic, this concept is auto and is defined via the standard's type_traits facilities and True.
