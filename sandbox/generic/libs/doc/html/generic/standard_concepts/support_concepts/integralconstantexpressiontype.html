IntegralConstantExpressionType

Description

Describes types that can be the type of an integral constant expression.

Definition

Table 1.7. IntegralConstantExpressionType Definition

BOOST_GENERIC_AUTO_CONCEPT

, (IntegralConstantExpressionType)( (typename) T )
, ( public (ScalarType< T >) (NonTypeTemplateParameterType< T >) )

concept IntegralConstantExpressionType<typename T>
  : ScalarType<T>, NonTypeTemplateParameterType<T> { }

// Requires: for every type T that is an integral type or enumeration type, a
// concept map IntegralConstantExpressionType<T> is implicitly defined in

Header

Section in N2914

The corresponding definition of IntegralConstantExpressionType may be found in the N2914 section [concept.support].

Differences From N2914

In N2914 this concept is explicit with concept maps implicitly defined for the implementation's arithmetic types. In Boost.Generic, this concept is auto and is defined via the standard's type_traits facilities and True.
