ClassType

Description

Describes class types (i.e., unions, classes, and structs).

Definition

Table 1.4. ClassType Definition

Boost.Generic Definition

N2914 Definition 

BOOST_GENERIC_AUTO_CONCEPT
( ( namespace boost, generic, std_concept )
, (ClassType)( (typename) T )
, ( requires True< std::is_class< T >::value || std::is_union< T >::value > )
)

concept ClassType<typename T> : ObjectType<T> { }

// Requires: for every type T that is a class type, a concept map ClassType<T>
// is implicitly defined in namespace std.

Header 
#include <boost/generic/std_concept/support_concepts/class_type.hpp>
Section in N2914

The corresponding definition of ClassType may be found in the N2914 section [concept.support].

Differences From N2914

In N2914 this concept is explicit with concept maps implicitly defined for the implementation's arithmetic types. In Boost.Generic, this concept is auto and is defined via the standard's type_traits facilities and True.
