DerivedFrom

Description

Describes types that are the same or that are unambiguously related by public inheritance.

Definition

concept DerivedFrom<typename Derived, typename Base> { }

// Requires: for every pair of class types (T, U), such that T is either the
// same as or publicly and unambiguously derived from U, a concept map
// DerivedFrom<T, U> is implicitly defined in namespace std.


Header 
#include <boost/generic/std_concept/support_concepts/derived_from.hpp>
Section in N2914

The corresponding definition of DerivedFrom may be found in the N2914 section [concept.support].

Differences From N2914

In N2914 this concept is explicit with concept maps implicitly defined for the implementation's arithmetic types. In Boost.Generic, this concept is auto and is defined via the standard's type_traits facilities and True.
