NonTypeTemplateParameterType

Description

Describes type that can be used as the type of a non-type template parameter.

Definition

Table 1.11. NonTypeTemplateParameterType Definition

Boost.Generic Definition

N2914 Definition 

BOOST_GENERIC_AUTO_CONCEPT
( ( namespace boost, generic, std_concept )
, (NonTypeTemplateParameterType)( (typename) T ), ( public VariableType< T > )
, ( requires True
             <    std::is_integral< T >::value
               || std::is_enum< T >::value
               || std::is_pointer< T >::value
               || std::is_reference< T >::value
                  // ToDo: Add archetype check as well
             >
  )
)


concept NonTypeTemplateParameterType<typename T> : VariableType<T> { }

// Requires: for every type T that can be the type of a non-type
// template-parameter, a concept map NonTypeTemplateParameterType<T> is
// implicitly defined in namespace std.


Header 
#include <boost/generic/std_concept/support_concepts/non_type_template_parameter_type.hpp>
Section in N2914

The corresponding definition of NonTypeTemplateParameterType may be found in the N2914 section [concept.support].

Differences From N2914

In N2914 this concept is explicit with concept maps implicitly defined for the implementation's arithmetic types. In Boost.Generic, this concept is auto and is defined via the standard's type_traits facilities and True.
