Class template overload behaves as an overloaded boost::function. Actually it wraps multiple boost::function objects of different call signatures. In this context N is equal to overload::extent and refers to the number of supported call signatures.

overload type traits
  1. template<unsigned int I>
    struct signature;
    Requires: The index I is such that 0<=I<N.
    Returns: signature<I>::type is the (I+1)-th call signature listed in the template argument list of this_type.

  2. template<unsigned int I>
    struct function;
    Requires: The index I is such that 0<=I<N.
    Returns: function<I>::type is the same as boost::function<S>, where S is the call signature signature<I>::type.

  3. template<typename Signature>
    struct index_of_signature;
    Returns: index_of_signature<Signature>::value is an integral constant that represents the position, starting from 0, of the call signature Signature in the template argument list of this_type if it is present and -1 otherwise .

  4. template<typename Signature>
    struct has_signature;
    Returns: has_signature<Signature>::value is a boolean constant equals to true if this_type supports the call signature Signature and false otherwise.

  5. template<typename Functor>
    struct shares_any_signature_with;
    Returns: shares_any_signature_with<Functor>::value is a boolean constant equals to true if this_type shares at least one call signature with the callable object type Functor.

overload public construct/copy/destruct
  1. overload();
    Post conditions: this->empty_all().
    Throws: Will not throw.

  2. overload(const this_type& f);
    Post conditions: Each object target is in the same state as the copied object target of f.
    Throws: Will not throw unless copying one of the object targets of f throws.

  3. template<typename Functor>
    overload(const Functor& f);
    Requires: f is a monomorphic callable object that is callable by *this.
    Post conditions: The object target related to S is a copy of f if f is not empty or this->empty<S>() if f is empty, where S is the call signature supported by f.

  4. template<typename F1, typename F2, ..., typename FN>
    overload(const F1& f1, const F2& f2, ..., const FN& fN);
    Requires: f1,f2,...,fN are monomorphic callable objects that are callable by *this.
    Post conditions: For each I=1,...N the object target related to S is a copy of fI if fI is not empty or this->empty<S>() if fI is empty, where S is the call signature supported by fI.

  5. this_type& operator=(const this_type& f);
    Post conditions: If copy construction does not throw, for each supported call signature S *this targets a copy of f's target related to S, if it has one, or is empty if f.empty<S>(). If copy construction does throw while copying the I-th object target then this->get<J>() is a copy of f.get<J> for J<I, this->get<J>() is not modified for J>I and finally this->empty<I>().

  6. ~overload();
    Effetcs: Destroys each non-empty target.
overload modifiers
  1. template<typename Functor>
    this_type& set(const Functor& f);
    Requires: f is a monomorphic callable object that is callable by *this.
    Post conditions: The object target related to S is a copy of f if f is not empty or this->empty<S>() if f is empty, where S is the call signature supported by f.

  2. template<typename F1, typename F2, ..., typename FN>
    this_type& set(const F1& f1, const F2& f2, ..., const FN& fN);
    Requires: f1,f2,...,fN are monomorphic callable objects that are callable by *this.
    Post conditions: For each I=1,...N the object target related to S is a copy of fI if fI is not empty or this->empty<S>() if fI is empty, where S is the call signature supported by fI.

  3. template<typename Signature, typename Functor>
    this_type& set(const Functor& f);
    Requires: Signature is a call signature supported by *this and f is a callable object that is callable by *this.
    Post conditions: The object target related to the call signature Signature is a copy of f if f is not empty or this->empty<Signature>() if f is empty.

  4. template<unsigned int I, typename Functor>
    this_type& set(const Functor& f);
    Requires: The index I is such that 0<=I<N and f is a callable object that is callable by *this.
    Post conditions: The object target related to the call signature S:=signature<I>::type is a copy of f if f is not empty or this->empty<S>() if f is empty.

  5. template<typename Functor>
    this_type& set_for_each_shared_signature(const Functor& f);
    Requires: f is a callable object that is callable by *this with any call signature shared by both *this and f.
    Post conditions: For each call signature S shared by both *this and f the object target related to S is a copy of f if f is not empty or this->empty<S>() if f is empty.

  6. void swap(this_type& f);
    Effetcs: For each supported call signature S interchanges the object targets of *this and f related to S

  7. template<typename Signature>
    void swap_function(boost::function<Signature>& f);
    Requires: Signature is a call signature supported by *this.
    Effetcs: Interchanges the object target of *this related to the call signature Signature with the object target of f.
  8. void clear_all();
    Post conditions: this->empty_all().

  9. template<typename Signature>
    void clear();
    Requires: Signature is a call signature supported by *this.
    Post conditions: this->empty<Signature>().

  10. template<unsigned int I>
    void clear();
    Requires: The index I is such that 0<=I<N.
    Post conditions: this->empty<I>().

overload capacity
  1. bool empty_all();
    Returns: false if *this has at least one call signature S such that !this->empty<S>() and true otherwise.

  2. bool empty_any();
    Returns: false if for each supported call signature S !this->empty<S>() and true otherwise.

  3. bool empty_none();
    Returns: false if *this has at least one call signature S such that this->empty<S>() and true otherwise.

  4. template<typename Signature>
    bool empty();
    Requires: Signature is a call signature supported by *this.
    Returns: false if this->get<Signature>() has a target and true otherwise.

  5. template<unsigned int I>
    bool empty();
    Requires: The index I is such that 0<=I<N.
    Returns: false if this->get<I>() has a target and true otherwise.

    Requires: Signature is a call signature supported by *this.
    Returns: a const reference to the unique embedded object of type boost::function<S> where S is the call signature signature<I>::type

  4. template<unsigned int I>

    Requires: The index I is such that 0<=I<N.
    Returns: a const reference to the unique embedded object of type boost::function<Signature>

Description

In the following context N is an integral constant between 1 and K=BOOST_OVERLOAD_LIMITS.

  1. template<typename F1 , typename F2, ..., typename FK>
    struct make_overload_type;
    Requires: F1, F2, ..., FK are callable object types with different call signatures or the detail::no_function type.
    Returns: make_overload_type<F1, F2, ..., FN>::type is the same type as overload<S1, S2, ..., SN> where SI is the call signature of the callable object type FI, for I=1,..N.

  2. template<typename F1 , typename F2, ..., typename FN>
    typename make_overload_type<F1, F2, ..., FN>::type
    make_overload(const F1& f1, const F2& f2, ...,const FN& fN);
    Requires: f1,f2,...,fN are monomorphic callable objects each supporting a different call signature.
    Returns: An overload object f that supports all and only the call signatures supported by f1, ..., fN and that for each I=1,...,N the object target related to the call signature signature<I>::type is a copy of fI.
