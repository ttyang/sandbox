Synopsis 

namespace boost {

    template <typename Graph>
    struct edge_key_trait
    {
        typedef void type;
    };

}  // namespace boost

Description

This metafunction returns the edge key type of Graph if it models the Associative Incidence Graph concept, or void if it does not. Specialize this metafunction for your custom graph type if you're designing it to be an Associative Incidence Graph.
