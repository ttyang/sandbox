Description

BGL representation of a multi-dimensional zeta grid, a rectangular grid that allows diagonal edges between vertices as well as horizontal and vertical ones. Two vertices are considered adjacent to each other if they are not at the same location and none of their corresponding coordinates in each dimension are different by more than one unit (taking wraparound into account).

The vector from a vertex to an adjacent one is the edge key associated with it. A two-dimensional zeta_grid_graph has eight distinct edge keys, corresponding to the eight cardinal and ordinal directions; a three-dimensional one has 26 edge keys; a zeta_grid_graph with N dimensions has 3 ^ N - 1 edge keys.

This data structure is an implicit graph, which means that its vertices and edges take up no internal memory.
