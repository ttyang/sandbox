Description

Models of this concept represent grid structures by matching their vertices to the corresponding grid points. Only vertices that are next to each other according to the grid structure have in-edges to and from each other.

Refinement of 

* Grid Graph
* Bidirectional Graph

Notation

Graph

A type that models the Bidirectional Grid Graph concept.

const_graph

An object of type Graph const&.

v

An object of type boost::graph_traits<Graph>::vertex_descriptor const&.

key

An object of type boost::edge_key_trait<Graph>::type const&.

distance

An unsigned integer type.

Associated Types

None beyond those defined by the Grid Graph and Bidirectional Graph concepts.

Valid Expressions and Semantics

In addition to the expressions defined in the Grid Graph and Bidirectional Graph concepts, the following expressions must be valid.

Runtime Complexity

Previous vertex 

const_graph.previous(v, key, distance)


boost::graph_traits<Graph>::vertex_descriptor

Equivalent to, but more efficient than: 

boost::graph_traits<Graph>::vertex_descriptor u = v;

for (
    BOOST_TYPEOF(distance) d = boost::initialized_value;
    d < distance;
    ++d
)
{
    u = const_graph.previous(u, key);
}

return u;

Linear with respect to the in-degree of v; amortized constant with respect to distance
