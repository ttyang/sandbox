Description

Models of this concept represent grid structures by matching their vertices to the corresponding grid points. Only vertices that are next to each other according to the grid structure have out-edges to and from each other.

Dimension wraparound query 

const_graph.wrapped(dimension)

bool

Returns true if two vertices on opposite ends of a grid line running along the specified dimension are adjacent to each other, false otherwise. A two-dimensional grid is considered a cylinder if it wraps around exactly one dimension, or a torus if it wraps around both dimensions.

Next vertex 

const_graph.next(v, key, distance)


boost::graph_traits<Graph>::vertex_descriptor

Equivalent to, but more efficient than: 

boost::graph_traits<Graph>::vertex_descriptor u = v;

for (
    BOOST_TYPEOF(distance) d = boost::initialized_value;
    d < distance;
    ++d
)
{
    u = const_graph.next(u, key);
}

return u;

Linear with respect to the out-degree of v; amortized constant with respect to distance
