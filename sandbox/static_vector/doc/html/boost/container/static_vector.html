Description

static_vector public construct/copy/destruct

  1. static_vector();

    Effects: Constructs an empty static_vector.

    Throws: Nothing.

    Complexity: Constant. 

  2. explicit static_vector(size_type count);

    Requires: count <= Capacity.

    Effects: Constructs a static_vector containing count default constructed Values.

    Throws: If Value's default constructor throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  3. static_vector(size_type count, value_type const & value);

    Requires: count <= Capacity.

    Effects: Constructs a static_vector containing count copies of value.

    Throws: If Value's copy constructor throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  4. template<typename Iterator> static_vector(Iterator first, Iterator last);

    Requires: distance(first, last) <= Capacity.

    Effects: Constructs a static_vector containing copy of a range [first, last).

    Throws: If Value's constructor taking a dereferenced Iterator throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  5. static_vector(static_vector const & other);

    Effects: Constructs a copy of other static_vector.

    Throws: If Value's copy constructor throws.

    Complexity: Linear. 

  6. template<std::size_t C, typename S> 
  static_vector(static_vector< value_type, C, S > const & other);

    Requires: other.size() <= Capacity.

    Effects: Constructs a copy of other static_vector.

    Throws: If Value's copy constructor throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  7. static_vector(static_vector && other);

    Effects: Move constructor. Moves Values stored in the other static_vector to this one.

    Throws: If boost::has_nothrow_move<Value>::value is true and Value's move constructor throws or if boost::has_nothrow_move<Value>::value is false and Value's copy constructor throws but only if use_memop_in_swap_and_move is false_type - default.

    Complexity: Linear. 

  8. template<std::size_t C, typename S> 
  static_vector(static_vector< value_type, C, S > static_vector< value_type, C, S > && other);

    Requires: other.size() <= Capacity.

    Effects: Move constructor. Moves Values stored in the other static_vector to this one.

    Throws: If boost::has_nothrow_move<Value>::value is true and Value's move constructor throws or if boost::has_nothrow_move<Value>::value is false and Value's copy constructor throws but only if use_memop_in_swap_and_move is false_type - default. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  9. static_vector& operator=(const static_vector & other);

    Effects: Copy assigns Values stored in the other static_vector to this one.

    Throws: If Value's copy constructor or copy assignment throws.

    Complexity: Linear. 

  10. template<std::size_t C, typename S> 
  static_vector& operator=(static_vector< value_type, C, S > const & other);

    Requires: other.size() <= Capacity.

    Effects: Copy assigns Values stored in the other static_vector to this one.

    Throws: If Value's copy constructor or copy assignment throws, If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  11. static_vector& operator=(static_vector && other);

    Effects: Move assignment. Moves Values stored in the other static_vector to this one.

    Throws: If boost::has_nothrow_move<Value>::value is true and Value's move constructor or move assignment throws, or if boost::has_nothrow_move<Value>::value is false and Value's copy constructor or copy assignment throws, but only if use_memop_in_swap_and_move is false_type - default.

    Complexity: Linear. 

  12. template<std::size_t C, typename S> 
  static_vector& 
  operator=(static_vector< value_type, C, S > static_vector< value_type, C, S > && other);

    Requires: other.size() <= Capacity.

    Effects: Move assignment. Moves Values stored in the other static_vector to this one.

    Throws: If boost::has_nothrow_move<Value>::value is true and Value's move constructor or move assignment throws, or if boost::has_nothrow_move<Value>::value is false and Value's copy constructor or copy assignment throws, but only if use_memop_in_swap_and_move is false_type - default. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  13. ~static_vector();

    Effects: Destructor. Destroys Values stored in this container.

    Throws: Nothing

    Complexity: Linear.

static_vector public member functions

  1. void swap(static_vector & other);

    Effects: Swaps contents of the other static_vector and this one.

    Throws: If boost::has_nothrow_move<Value>::value is true and Value's move constructor or move assignment throws, or if boost::has_nothrow_move<Value>::value is false and Value's copy constructor or copy assignment throws, but only if use_memop_in_swap_and_move and use_optimized_swap are false_type - default.

    Complexity: Linear. 

  2. template<std::size_t C, typename S> 
  void swap(static_vector< value_type, C, S > & other);

    Requires: other.size() <= Capacity && size() <= other.capacity().

    Effects: Swaps contents of the other static_vector and this one.

    Throws: If boost::has_nothrow_move<Value>::value is true and Value's move constructor or move assignment throws, or if boost::has_nothrow_move<Value>::value is false and Value's copy constructor or copy assignment throws, but only if use_memop_in_swap_and_move and use_optimized_swap are false_type - default. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  3. void resize(size_type count);

    Requires: count <= Capacity.

    Effects: Inserts or erases elements at the end such that the size becomes count. New elements are default constructed.

    Throws: If Value's default constructor throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  4. void resize(size_type count, value_type const & value);

    Requires: count <= Capacity.

    Effects: Inserts or erases elements at the end such that the size becomes count. New elements are copy constructed from value.

    Throws: If Value's copy constructor throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  5. void reserve(size_type count);

    Requires: count <= Capacity.

    Effects: This call has no effect because the Capacity of this container is constant.

    Throws: If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  6. void push_back(value_type const & value);

    Requires: size() < Capacity.

    Effects: Adds a copy of value at the end.

    Throws: If Value's copy constructor throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Constant. 

  7. void push_back(value_type && value);

    Requires: size() < Capacity.

    Effects: Moves value to the end.

    Throws: If Value's move constructor throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Constant. 

  8. void pop_back();

    Requires: !empty().

    Effects: Destroys last value and decreases the size.

    Throws: Nothing by default.

    Complexity: Constant. 

  9. iterator insert(iterator position, value_type const & value);

    Requires: position must be a valid iterator of *this in range [begin(), end()].

    Effects: Inserts a copy of element at position.

    Throws: If Value's copy constructor or copy assignment throws or if Value's move constructor or move assignment throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Constant or linear. 

  10. iterator insert(iterator position, value_type && value);

    Requires: position must be a valid iterator of *this in range [begin(), end()] and size() < Capacity.

    Effects: Inserts a move-constructed element at position.

    Throws: If Value's move constructor or move assignment throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Constant or linear. 

  11. iterator insert(iterator position, size_type count, value_type const & value);

    Requires: position must be a valid iterator of *this in range [begin(), end()] and size() + count <= Capacity.

    Effects: Inserts a count copies of value at position.

    Throws: If Value's copy constructor or copy assignment throws or if Value's move constructor or move assignment throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  12. template<typename Iterator> 
  iterator insert(iterator position, Iterator first, Iterator last);

    Requires: position must be a valid iterator of *this in range [begin(), end()] and distance(first, last) <= Capacity.

    Effects: Inserts a copy of a range [first, last) at position.

    Throws: If Value's constructor and assignment taking a dereferenced Iterator throws or if Value's move constructor or move assignment throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Linear. 

  13. iterator erase(iterator position);

    Requires: position must be a valid iterator of *this in range [begin(), end()).

    Effects: Erases Value from position.

    Throws: If Value's move assignment throws.

    Complexity: Linear. 

  14. iterator erase(iterator first, iterator last);

    Requires: first and last must define a valid range, iterators must be in range [begin(), end()].

    Effects: Erases Values from a range [first, last).

    Throws: If Value's move assignment throws.

    Complexity: Linear. 

  15. template<typename Iterator> void assign(Iterator first, Iterator last);

    Requires: distance(first, last) <= Capacity.

    Effects: Assigns a range [first, last) of Values to this container.

    Throws: If Value's copy constructor or copy assignment throws,

    Complexity: Linear. 

  16. void assign(size_type count, value_type const & value);

    Requires: count <= Capacity.

    Effects: Assigns a count copies of value to this container.

    Throws: If Value's copy constructor or copy assignment throws.

    Complexity: Linear. 

  17. template<class... Args> void emplace_back(Args &&... args);

    Requires: size() < Capacity.

    Effects: Inserts a Value constructed with std::forward<Args>(args)... in the end of the container.

    Throws: If in-place constructor throws or Value's move constructor throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Constant. 

  18. template<class... Args> iterator emplace(iterator position, Args &&... args);

    Requires: position must be a valid iterator of *this in range [begin(), end()] and size() < Capacity.

    Effects: Inserts a Value constructed with std::forward<Args>(args)... before position

    Throws: If in-place constructor throws or Value's move constructor or move assignment throws. If the error handler throws in capacity check (nothing by default).

    Complexity: Constant or linear. 

  19. void clear();
