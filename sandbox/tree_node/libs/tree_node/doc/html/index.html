Rationale

Over the course of reimplementing a decision tree learning algorithm to use modern C++ techniques, the need for a generic tree library became evident. The in-house data structures in use at the time were undergoing modification as well. It was found that the core operations occurred at the node level rather than at the tree level, which stored only high-level algorithms. Efforts went underway to retrofit these algorithms to use iterators and to further factor out generic data structures from the decision tree components and other application-specific utilities. This library represents the Boostified culmination of these efforts.
