traversal_state

Synopsis 

namespace boost { namespace tree_node {

    enum traversal_state
    {
        no_traversal
      , pre_order_traversal
      , post_order_traversal
      , breadth_first_traversal
      , in_order_traversal
    };
}}  // namespace boost::tree_node

Description

This enumeration type is the return type of the dereferenceability expression defined by each iterator type that this library provides. The no_traversal value corresponds to false in a boolean context, enabling the iterator types to fulfill their runtime requirements for dereferenceability. The pre_order_traversal and post_order_traversal values are significant because the depth_first_iterator traverses each node twice, so algorithms must be able to distinguish which state the iterator is in when it traverses a tree node. The breadth_first_traversal and in_order_traversal values are defined for completeness.
