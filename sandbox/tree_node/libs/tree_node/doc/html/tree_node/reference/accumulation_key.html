accumulation_key

Synopsis 

namespace boost { namespace tree_node {


    template <
        typename Key = data_key
      , typename Tag = ::boost::accumulators::tag::sum
      , typename IncludesAllDescendants = ::boost::mpl::true_
      , typename IncludesRoot = ::boost::mpl::true_
      , typename Value = void
    >
    struct accumulation_key
    {
        typedef Key key;
        typedef Tag tag;
        typedef IncludesAllDescendants includes_all_descendants;
        typedef IncludesRoot includes_root;
        typedef Value value;
    };

    typedef accumulation_key<> default_accumulation_key;
}}  // namespace boost::tree_node

Description

Use this type to access statistical data stored in Tree Node objects of types ultimately inheriting from tree_node::with_accumulation_base.

Where defined 

#include <boost/tree_node/key/accumulation.hpp>

To register under Boost.Typeof: 

#include <boost/typeof/boost/tree_node/key/accumulation.hpp>

Template Parameters

Parameter

Description

Requirements

Default

Key

The type used to access the data to be accumulated.

tree_node::result_of::has_key<Node,Key> must evaluate to boost::mpl::true_ for some Node node storing the desired data.

tree_node::data_key

Tag

The type of the actual computation.

It must be a valid Boost.Accumulators feature.

accumulators::tag::sum

IncludesAllDescendants

Configures the tree_node::with_accumulation_base to include all descendants of a target node for statistical evaluation if set to boost::mpl::true_; otherwise, only the node's immediate children will be included.

It must model the Boolean Integral Constant concept.

boost::mpl::true_

IncludesRoot

Configures the tree_node::with_accumulation_base to include a target node along with its children/descendants for statistical evaluation if set to boost::mpl::true_; otherwise, the node will be excluded from evaluation.

It must model the Boolean Integral Constant concept.

boost::mpl::true_

Value

The type of the accumulated result. If left unspecified, the tree_node::with_accumulation_base will use the type of the data to be accumulated.

void
