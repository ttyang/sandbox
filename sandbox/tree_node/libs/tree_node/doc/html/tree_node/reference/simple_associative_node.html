Description

This class template is suitable for classes or algorithms that need to map keys to child nodes in a tree node data structure that must be exposed to external code. Objects of this type must be stored by simple_associative_node::pointer and can only be instantiated through the static creation methods that this class template provides. Their children can be accessed by their associated keys for fast lookup.

Important
