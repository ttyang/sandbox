Description

This iterator traverses the descendants of the specified root node in either pre-order or reverse post-order fashion, e.g. given the following tree representation: 

      A
      |
  C---+---B
  |       |
+-+-+   +-+-+
|   |   |   |
D   E   F   G

This iterator will traverse the tree nodes in the following order: 

C D E B F G

Requirements

Default

Node

The type of a tree node.

It must model the Tree Node concept, but it can be const-qualified.

IsReverse

Configures the pre_order_descendant_iterator to traverse tree nodes in reverse post-order fashion if equivalent to boost::mpl::true_; otherwise, the nodes will be traversed in pre-order fashion.

It must model the Boolean Integral Constant concept.

boost::mpl::false_
