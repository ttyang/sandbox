BOOST_TREE_NODE_EMPLACEMENT_CTOR_FWD_DECL

Synopsis 

#define BOOST_TREE_NODE_EMPLACEMENT_CTOR_FWD_DECL(z, n, Type)                \
        BOOST_TREE_NODE_EMPLACEMENT_CTOR_HEADER(z, n, Type);                 \

Description

Defined only if the compiler does not support perfect forwarding, and meant to be used in the public scope of a Base Tree Node, this macro expands to the forward declaration of a constructor that will emulate it. Uses BOOST_TREE_NODE_EMPLACEMENT_CTOR_HEADER.

Usage 

BOOST_PP_REPEAT(
    BOOST_CONTAINER_MAX_CONSTRUCTOR_PARAMETERS
  , BOOST_TREE_NODE_EMPLACEMENT_CTOR_FWD_DECL
  , Type
)

Arguments 

* /z/ -- The next available BOOST_PP_REPEAT dimension.
* /n/ -- The number of arguments that the current iteration's constructor
  will accept.
* /Type/ -- The name of the enclosing type.

Where defined 

#include <boost/tree_node/preprocessor.hpp>
