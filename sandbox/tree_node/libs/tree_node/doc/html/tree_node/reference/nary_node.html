nary_node<T,Selector>

Description

This class template is suitable for classes or algorithms that require a tree node data structure that can handle an arbitrary number of children. Objects of this type can be stored by value.

Where defined 

#include <boost/tree_node/nary_node.hpp>

Template Parameters

Parameter

Description

Requirements

Default

T

The type of the object to be stored in an nary_node.

Selector

The type that determines the child container type.

is_recursive_selector<Selector> must return boost::mpl::true_.

ptr_dequeS

Model of 

* N-ary Tree Node
* Default Constructible if `T` is Default Constructible
* Copy Constructible if `T` is Copy Constructible
* Assignable if `T` is Copy Constructible and Movable
* Movable if `T` is Movable
