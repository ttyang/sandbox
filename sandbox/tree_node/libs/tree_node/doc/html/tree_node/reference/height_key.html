height_key

Synopsis 

namespace boost { namespace tree_node {

    struct height_key
    {
    };
}}  // namespace boost::tree_node

Description

Use this type to access the height of a Tree Node object, or the magnitude of the difference in ply between it and its deepest descendant. The Tree Node type must ultimately inherit from tree_node::with_height_base.

Where defined 

#include <boost/tree_node/key/height.hpp>

To register under Boost.Typeof: 

#include <boost/typeof/boost/tree_node/key/height.hpp>
