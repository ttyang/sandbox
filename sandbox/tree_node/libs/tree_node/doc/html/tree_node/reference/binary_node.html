binary_node<T>

Description

This class template is suitable for classes or algorithms that require a binary tree node data structure. Objects of this type can be stored by value.

Where defined 

#include <boost/tree_node/binary_node.hpp>

Template Parameters

Parameter

Description

Requirements

Default

T

The type of the object to be stored in a binary_node.

Model of 

* Binary Tree Node
* Default Constructible if `T` is Default Constructible
* Copy Constructible if `T` is Copy Constructible
* Assignable if `T` is Copy Constructible and Movable
* Movable if `T` is Movable
