BOOST_TREE_NODE_EMPLACEMENT_CTOR_BASE_FWD

Synopsis 

#define BOOST_TREE_NODE_EMPLACEMENT_CTOR_BASE_FWD(z, n, Base)                \
      : Base(                                                                \
            BOOST_PP_CAT(BOOST_PP_ENUM_, z)(                                 \
                n                                                            \
              , BOOST_CONTAINER_PP_PARAM_FORWARD                             \
              , _                                                            \
            )                                                                \
        )                                                                    \

Description

Defined only if the compiler does not support perfect forwarding, this macro expands to the first part of the initializer list of a constructor that will emulate perfect forwarding. Used by BOOST_TREE_NODE_EMPLACEMENT_CTOR_INLINE_HEADER.

Where defined 

#include <boost/tree_node/preprocessor.hpp>
