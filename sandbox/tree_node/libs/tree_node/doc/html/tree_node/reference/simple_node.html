Description

This class template is suitable for classes or algorithms that require a tree node data structure that must be exposed to external code. Objects of this type must be stored by simple_node::pointer and can only be instantiated through the static creation methods that this class template provides. Their children can be accessed by offsetting the start of the range with a numerical index for fast lookup.
