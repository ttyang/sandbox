Description

This iterator traverses the specified root node and all its descendants in in-order fashion, e.g. given the following tree representation: 

      A
      |
  C---+---B
  |       |
+-+-+   +-+-+
|   |   |   |
D   E   F   G

The tree nodes will be traversed forward in the following order: 

D C E A F B G

This iterator can also be used to traverse the tree nodes backwards: 

G B F A E C D

Important

This iterator works only on Binary Tree Node models.
