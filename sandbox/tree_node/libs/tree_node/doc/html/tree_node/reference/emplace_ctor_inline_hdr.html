BOOST_TREE_NODE_EMPLACEMENT_CTOR_INLINE_HDR

Synopsis 

#define BOOST_TREE_NODE_EMPLACEMENT_CTOR_INLINE_HEADER(z, n, Tuple)          \
        BOOST_TREE_NODE_EMPLACEMENT_CTOR_HEADER(                             \
            z                                                                \
          , n                                                                \
          , BOOST_PP_TUPLE_ELEM(2, 0, Tuple)                                 \
        )                                                                    \
        BOOST_TREE_NODE_EMPLACEMENT_CTOR_BASE_FWD(                           \
            z                                                                \
          , n                                                                \
          , BOOST_PP_TUPLE_ELEM(2, 1, Tuple)                                 \
        )                                                                    \

Description

Defined only if the compiler does not support perfect forwarding, this macro expands to the header of a constructor that emulates it along witht the first part of its initialization list. Uses BOOST_TREE_NODE_EMPLACEMENT_CTOR_BASE_FWD; used by BOOST_TREE_NODE_EMPLACEMENT_CTOR_INLINE_DEF.

Where defined 

#include <boost/tree_node/preprocessor.hpp>
