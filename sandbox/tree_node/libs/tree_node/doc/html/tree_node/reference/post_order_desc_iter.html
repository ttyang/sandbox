Description

This iterator traverses the descendants of the specified root node in post-order fashion, e.g. given the following tree representation: 

      A
      |
  C---+---B
  |       |
+-+-+   +-+-+
|   |   |   |
D   E   F   G

This iterator will traverse the tree nodes in the following order: 

D E C F G B
