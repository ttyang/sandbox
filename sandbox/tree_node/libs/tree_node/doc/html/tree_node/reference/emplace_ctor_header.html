BOOST_TREE_NODE_EMPLACEMENT_CTOR_HEADER

Synopsis 

#define BOOST_TREE_NODE_EMPLACEMENT_CTOR_HEADER(z, n, Type)                  \
        BOOST_PP_EXPR_IF(n, template <)                                      \
            BOOST_PP_ENUM_PARAMS_Z(z, n, typename P)                         \
        BOOST_PP_EXPR_IF(n, >)                                               \
        BOOST_PP_EXPR_IIF(BOOST_PP_EQUAL(n, 1), explicit)                    \
        Type(                                                                \
            BOOST_PP_CAT(BOOST_PP_ENUM_, z)(                                 \
                n                                                            \
              , BOOST_CONTAINER_PP_PARAM_LIST                                \
              , _                                                            \
            )                                                                \
        )                                                                    \

Description

Defined only if the compiler does not support perfect forwarding, this macro expands to the header of a constructor that will emulate it. Used by BOOST_TREE_NODE_EMPLACEMENT_CTOR_FWD_DECL and BOOST_TREE_NODE_EMPLACEMENT_CTOR_INLINE_HEADER.

Where defined 

#include <boost/tree_node/preprocessor.hpp>
