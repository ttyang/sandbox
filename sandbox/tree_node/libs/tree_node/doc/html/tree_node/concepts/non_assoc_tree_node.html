Non-Associative Tree Node

Description

Models of this concept provide access to their child nodes via iterator dereferencing.

Refinement of

Tree Node

Associated Types

No additional types beyond those defined in the Tree Node concept; however, the requirements for the child iterator types are strengthened: 

* The value type of `Node::child_iterator` must be `Node::pointer`.
* The value type of `Node::const_child_iterator` must be
  `Node::const_pointer`.

Valid Expressions and Semantics

None beyond those defined in the Tree Node concept.
