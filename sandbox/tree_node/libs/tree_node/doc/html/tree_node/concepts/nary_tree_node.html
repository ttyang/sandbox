N-ary Tree Node

Description

Models of this concept store arbitrary numbers of children and provide simple child insertion expressions.

Refinement of

Non-Associative Tree Node

Notation

Node

A type that models the N-ary Tree Node concept.

node

An object of type Node::pointer.

const_node

An object of type Node::const_pointer.

data

An object of type Node::traits::data_type const&.

Associated Types

No additional types beyond those defined in the Non-Associative Tree Node concept.

Valid Expressions and Semantics

In addition to the expressions defined in the Non-Associative Tree Node concept, the following expressions must be valid.
