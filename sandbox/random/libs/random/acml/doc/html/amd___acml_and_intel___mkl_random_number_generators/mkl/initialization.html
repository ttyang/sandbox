Initialization

Apart from default construction the engines can be initialized with a seed value. Using construction by seed sequence or by an iterator range, multiple seed values can be taken into account. The number of accepted values as well as their interpretation differs for the different engines. For a detailed description please refer to the Intel® VSL notes.
