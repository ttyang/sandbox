Macro CONTRACT_DESTRUCTOR_BODY

CONTRACT_DESTRUCTOR_BODY — Macro used to name the body of destructors.

Synopsis

// In header: <contract/body.hpp>

CONTRACT_DESTRUCTOR_BODY(class_type, destructor_name)

Description

This macro is used to name the body of destructors when the body is defined separately from the contract declaration. Destructors with contracts are declared using the CONTRACT_DESTRUCTOR macro.

Parameters:

class_type The syntax for the class type is explained in the Grammar section (for class templates, this type is qualified with the template parameters).
destructor_name This is the class name prefixed by the tilde symbol ~ and its syntax is explained in the Grammar section (for class templates, this name is not qualified with the template parameters).
