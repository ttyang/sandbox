Struct template strip

boost::dispatch::meta::strip

Synopsis

// In header: <boost/dispatch/meta/strip.hpp>

template<typename T  // Type to strip from reference and cv-qualifier.
         > 
struct strip {
  // types
  typedef T type;
};

Description

Removes all reference and cv-qualifier from a given type T.

:

For any given type T,

 typedef boost::dispatch::meta::strip<T>::type r;

is equivalent to:

 typedef remove_cv< remove_reference<T>::type >::type r;
