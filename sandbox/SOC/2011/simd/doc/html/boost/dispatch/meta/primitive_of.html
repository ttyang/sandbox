Struct template primitive_of

boost::dispatch::meta::primitive_of

Synopsis

// In header: <boost/dispatch/meta/primitive_of.hpp>

template<typename Hierarchizable  // Type to extract a primitive from.
         > 
struct primitive_of {
  // types
  typedef Hierarchizable type;
};

Description

For a given Hierarchizable type T, returns the undelying type used to define T

Models: 

Semantic: For a given Hierarchizable type T,

 typedef primitive_of<T>::type r;

is defined so that

 is_same<T, apply<factory_of<T>,r>::type>::value == true
