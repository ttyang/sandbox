boost::dispatch::meta::is_fundamental

Description

Checks if a given type is a fundamental type (i.e any scalar type natively supported by C++).

Model: 

For any type T 

 typedef is_fundamental<T>::type r;

evaluates to if T hierarchy inherits from fundamental_<T> and to otherwise
