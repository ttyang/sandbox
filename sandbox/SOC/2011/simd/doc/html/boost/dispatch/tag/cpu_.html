Struct cpu_

boost::dispatch::tag::cpu_

Synopsis

// In header: <boost/dispatch/functor/meta/hierarchy.hpp>


struct cpu_ : public boost::dispatch::tag::formal_ {
};

Description

Defines the CPU based evaluation context for functors. This context is used when no specific architecture informations is available or required by a functor. Functors specialization under the cpu_ context is usually used as common, architecture independant implementation.
