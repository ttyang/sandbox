Struct template call

boost::dispatch::meta::call

Synopsis

// In header: <boost/dispatch/functor/meta/call.hpp>

template<typename Sig, 
         typename EvalContext = typename default_site< typename  boost::function_types::                                                        result_type<Sig>::type                                            >::type> 
struct call {
};

Description

call is a that, for a given Tag, EvalContext and argument lists ...Args, computes the return type of functor<Tag,EvalContext>::operator()(Args...).

Models: 

Semantics: For a given Tag, EvalContext and argument lists ...Args,

 typedef call<Tag(Args...),EvalContext>::type type;

is equivalent to: 

 typedef decltype(functor<Tag,EvalContext>(declval<Args...>())) type;
