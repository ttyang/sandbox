boost::dispatch::meta::is_floating_point

};

Description

For any type T, checks if its primitive type is a floating-point type.

Models: 

For any type T,

 typedef is_floating<T>::type type;

is equivalent to

 typedef behave_as<boost::is_floating_point<boost::mpl::_>
                  , strip<T>::type
                  > type;
