Description

functor is a generic, concept-dispatched polymorphic function object. Every function in Boost.Dispatch are defined in term of implementing a call strategy for functor. The main rationale is to centralize and homogenize the way functions with multiple possible implementation with respect to types, architectures and compilation phases are handled.
