Struct template make_functor

boost::dispatch::make_functor

Synopsis

// In header: <boost/dispatch/functor/meta/make_functor.hpp>

template<typename Tag, typename Dummy = void> 
struct make_functor {
  // types
  typedef functor< Tag > type;
};

Description

This meta-function builds a concrete functor type from a given function Tag. It's main purpose is to delay the instanciation of functor type in dependant context to satisfy the two-phase look-up rule enforced by clang and g++-4.6 and superior.

Models: 

Semantic: 

For any Tag : 

 typedef make_functor<Tag>::type type;

is equivalent to

 typedef functor<Tag> type;
