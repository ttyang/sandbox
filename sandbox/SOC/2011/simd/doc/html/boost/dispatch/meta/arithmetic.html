Struct arithmetic

boost::dispatch::meta::arithmetic

Synopsis

// In header: <boost/dispatch/meta/arithmetic.hpp>


struct arithmetic {
  // member classes/structs/unions
  template<typename This, typename A0> 
  struct result<This(A0)> {
    // types
    typedef A0 type;
  };
};

Description

For a list of types T0,...,Tn, ref arithmetic computes the type obtained by the recursive application of decltype(0 ? declval<T0>() : declval<T1>())

For types T0,...,Tn,

 typedef result_of<arithmetic(T0,...,Tn)>::type type

is equivalent to:

 typedef boost::common_type<T0,...,TN>::type type;
