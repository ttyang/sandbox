Struct template is_iterator

boost::dispatch::meta::is_iterator

Synopsis

// In header: <boost/dispatch/meta/is_iterator.hpp>

template<typename T, typename Enable = void> 
struct is_iterator {
};

Description

Checks if a given type models the concept.

For any type T,

  typedef boost::dispatch::meta::is_iterator<T>::type v;

evaluates to if T is a type modeling the concept. Such types includes standard conforming iterator classes and non-function pointers.
