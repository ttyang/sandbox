Macro BOOST_DISPATCH_FUNCTOR_CALL_REPEAT

BOOST_DISPATCH_FUNCTOR_CALL_REPEAT

Synopsis

// In header: <boost/dispatch/functor/preprocessor/call.hpp>

BOOST_DISPATCH_FUNCTOR_CALL_REPEAT(N)

Description

Generates a boost::dispatch::ext::call specialization operator() prototype. In this function, arguments are defined as a0,...,an-1 of the same type A0.

 BOOST_DISPATCH_FUNCTOR_CALL_REPEAT(3)
 {
   return a0 + a1/a2;
 }
