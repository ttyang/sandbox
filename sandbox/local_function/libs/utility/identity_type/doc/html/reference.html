Header <boost/utility/identity_type.hpp>

Identity macros are used to pass expressions with commas (not already wrapped within round parenthesis) as macros parameters.

Note: The only way (at least as far as the authors know) to pass an arbitrary number of commas within macro parameters without explicitly indicating the number of commas (and without using variadic macros which are not part of the C++03 standard) is to wrap the expression within round parenthesis that are not replaced by macro expansion and then to remove the extra parenthesis at compile-time (for type expressions) or at run-time (for value expressions). This is what these macros do.
