Macro BOOST_LOCAL_FUNCTION_CONFIG_FUNCTION_ARITY_MAX

BOOST_LOCAL_FUNCTION_CONFIG_FUNCTION_ARITY_MAX — Maximum number of function parameters supported for the local functions.

Synopsis

// In header: <boost/local_function/config.hpp>

BOOST_LOCAL_FUNCTION_CONFIG_FUNCTION_ARITY_MAX

Description

If programmers leave this configuration macro undefined, its default value is 5.

This only applies to the number of local function parameters and not to the number of bound variables in scope (the limit on the number of bound variables is instead the maximum size allowed for a Boost.Preprocessor sequence).

Warning: Increasing this number will increase compilation time.

See: Tutorial section, {Getting_Started, Getting Started} section.
