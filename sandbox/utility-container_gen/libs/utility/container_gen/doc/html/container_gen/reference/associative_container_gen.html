associative_container_gen

Synopsis 

template <typename Selector>
struct associative_container_gen
{
    template <typename Key, typename Mapped = void>
    struct apply
    {
        // typedef ... type;
    };
};

Description

The raw_associative_node and simple_associative_node data structures use this metafunction class to determine the storage type that will associate key objects with child nodes. This library provides specializations of this metafunction class for each selector it provides whose resulting storage type models the Associative Container concept.

It is possible to nest containers via associative_container_gen, e.g.: 

typedef boost::mpl::apply_wrap2<
            associative_container_gen<mapS>
          , boost::mpl::apply_wrap1<associative_container_gen<setS>,char>::type
          , std::string
        >::type
        MapOfCharSets2Strings;

Important

This component depends on Boost.TypeTraitsOperators, which has been recently accepted into Boost but is currently not part of an official release. For now, you must perform a Subversion checkout from the SVN Trac.

Where defined 

#include <boost/utility/associative_container_gen.hpp>
