is_unordered_selector

Synopsis 

namespace boost {

    template <typename Selector>
    struct is_unordered_selector : ::boost::mpl::false_
    {
    };

}  // namespace boost

Description

Returns boost::mpl::true_ if the storage type to be returned by container_gen is an instantiation of either std::tr1::unordered_set or std::tr1::unordered_multiset, or if the storage type to be returned by associative_container_gen is an instantiation of either std::tr1::unordered_map or std::tr1::unordered_multimap, boost::mpl::false_ otherwise.

Where defined 

#include <boost/utility/is_unordered_selector.hpp>
