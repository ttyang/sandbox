Synopsis 

namespace boost {

    template <typename InputGraph>
    bool has_eulerian_cycle(InputGraph const& input_graph);

}  // namespace boost

Description

For an undirected graph, returns true if every vertex has an even out-degree and if all of its vertices belong to a single connected component; for a directed graph, returns true if the in-degree of every vertex is equal to its out-degree and if all of its vertices belong to a single strongly connected component; returns false otherwise.
