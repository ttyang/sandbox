Synopsis 

namespace boost {

    template <typename InputGraph, typename ComponentMap>
    typename ::boost::property_traits<ComponentMap>::value_type
        weak_components(InputGraph const& graph, ComponentMap component_map);

}  // namespace boost

Description

This algorithm computes the weakly connected components of a directed graph by computing the connected components of an isomorphic undirected graph. Returns the total number of components it computes.

Where defined 

#include <boost/graph/weak_components.hpp>

Type Requirements 

* `InputGraph` must model the Incidence Graph and Vertex List Graph
  concepts.
* The traversal category type of `InputGraph` must inherit from
  `boost::directed_category_tag`.
* `ComponentMap` must model the Read/Write Property Map concept.
* The key type of `ComponentMap` must be the same as the
  vertex descriptor type of `InputGraph`.
* The value type of `ComponentMap` must be an unsigned integral type.

Parameters

Parameter

Description

input_graph

A directed graph.

component_map

A property map that stores the output of the algorithm. Each vertex will be associated with an integer label. All vertices with the same integer label belong to the same weak component.

Complexity

The time complexity is O(V + E).
