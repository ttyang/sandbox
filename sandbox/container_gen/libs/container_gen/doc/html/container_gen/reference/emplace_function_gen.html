emplace_function_gen

Synopsis 

namespace boost {

    template <typename Selector>
    struct emplace_function_gen
    {
        // typedef ... type;
    };
}  // namespace boost

Description

Returns a Default Constructible n-ary function object type whose first argument type is a reference to a container. In practice, the type of this container must be produced by container_gen when given the same selector template argument. The rest of the function object's arguments, if any, will be forwarded either to the container's emplacement method (if has_emplace_member_function_selector returns boost::mpl::true_ for the given selector) or to the value type's matching constructor.

If the container is a mutable Boost.Heap object, then the call operator's return value is the handle returned by the container's emplacement function call. If the container is a Boost.Heap object that is not mutable, then the call operator's return type is void. Otherwise, the call operator's return value is a std::pair whose second element evaluates to true upon successful insertion and whose first element is an iterator pointing to the element inserted.

The return type of this metafunction also defines an indexing operator that accepts a reference to a container and returns a proxy object that allows emplacement function calls to be chained together, much like Boost.Assign.

Where defined 

#include <boost/container_gen/emplace_function_gen.hpp>

Example 

template <typename Selector>
void example()
{
    typename container_gen<Selector,std::string>::type c1, c2, c3;
    typename emplace_function_gen<Selector>::type emplacer;

    BOOST_AUTO_TPL(result1, emplacer(c1, "Hello,"));
    BOOST_AUTO_TPL(result2, emplacer(c2, "Hello,"));
    BOOST_ASSERT(result1 == result2);

    result1 = emplacer(c1, "world!");
    result2 = emplacer(c2, "world!");
    BOOST_ASSERT(result1 == result2);
    BOOST_ASSERT(boost::range::equal(c1, c2));

    emplacer[c3]("Hello,")("world!");
    BOOST_ASSERT(boost::range::equal(c1, c3));
}

int main()
{
    example<boost::vector_selector<boost::mpl::true_> >();
    return 0;
}
