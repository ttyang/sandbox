Function Synopsis 

namespace boost {

    template <typename T>
    T& get_reference()
    {
        return ::boost::optional<T>().get();
    }
}  // namespace boost

Function Description

Allows a BOOST_TYPEOF or BOOST_TYPEOF_TPL expression to obtain a reference to the specified type without requiring the type to be Default Constructible.
