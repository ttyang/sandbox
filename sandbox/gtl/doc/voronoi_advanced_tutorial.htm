Voronoi Advanced Tutorial

This tutorial consists of two parts. The first one provides three examples of a real world problems that default configuration of Voronoi library solves. By default configuration we mean the one that accepts signed 32-bit integer coordinates and outputs floating-point (64-bit double) coordinates. We provide those examples to convience even the most sceptical users that they probably don't need to configure library for higher-precision input or output coordinate types. However if the posed problem really requires those, fully featured configuration of both input and output coordinate types is provided in the second part of this tutorial.

Three Examples

In three chapters below we explain three different areas of application of Voronoi diagram. Each of the examples covers following topics:
  • Problem statement
  • Application of Voronoi diagram
  • Discretization of input geometries
  • Precision of output geometries
