A basic, heap-allocated Bloom filter implemented using the two-hash technique. It is best used when:

  • Storage requirements exceed stack space.
  • You wish to generate many hash values conveniently.

Template Parameters Reference

The type used for all Bloom filter operations.

The number of hash values to generate.

Currently unused. See here.
