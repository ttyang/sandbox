Version history

v0.6.5 (August 3rd)

  • Documentation Update
    • Updated reference - basic_bloom_filter, dynamic_bloom_filter, counting_bloom_filter
    • Updated tutorial - added information regarding new exception support for counting_bloom_filter
    • Updated history

v0.6.0 (August 2nd)

  • Implementation Update
    • Refactored counting_apply_hash - LOC cut in half, also usable for future dynamic_counting_bloom_filter.
  • API Changes
    • counting_bloom_filter - added various typedefs and a few metadata functions. More specifically:
      • Added bits_per_bin()
      • Added bins_per_slot()
      • Added mask()
      • Added num_bins()
      • Changed operator|=() -> experimental_union_assign
      • Changed operator|() -> experimental_union
      • Changed operator&=() -> experimental_intersect_assign
      • Changed operator&() -> experimental_intersect
      • Added restraint on BitsPerBin - must be greater than 0
      • Added restraint on BitsPerBin - must be <= sizeof(block_type)
  • Testing Update
    • Greatly expanded counting_bloom_filter tests. New coverage:
      • Ensured all valid BitsPerBin sizes compile
      • Ensured that many valid block_types compile
      • Added tests for falsePositiveRate()
      • Added tests for union/intersection
      • Added more tests for exception triggering
      • Made countMulti test more comprehensive - tested on all valid bitsPerBin values

v0.5.0 (July 26nd)

  • Implementation Update
    • Refactored apply_hash - both basic_bloom_filter and dynamic_bloom_filter use same apply_hash meta-function.
  • API Changes
    • basic_bloom_filter - added [this_type, bitset_type] typedef
    • dynamic_bloom_filter - added [this_type, bitset_type] typedef

v0.4.4 (July 24nd)

  • New!
    • murmurhash3 Hasher
  • API Changes
    • Added exception classes for bin_underflow and bin_overflow
    • bloom_filter -> basic_bloom_filter
    • bloom.hpp -> basic_bloom_filter.hpp
    • dynamic_bloom.hpp -> dynamic_bloom_filter.hpp
    • counting_bloom.hpp -> counting_bloom_filter.hpp
    • namespace bloom_filter -> namespace bloom_filters
  • Documentation Changes
    • Restructured tutorial into various modules
    • Added reference for counting_bloom_filter
    • Updated bibliography - restructured into three sections
    • Updated design - restructured into a section per Bloom filter variant

v0.4.0 (July 2nd)

  • New!
    • Compiling counting Bloom filter available
  • Test Changes
    • Added regression suite for counting Bloom filter

v0.3.0 (July 2nd)

  • New!
    • Dynamic Bloom filter completed.
    • Counting Bloom filter interface added to repository.
  • Test Changes
    • Added regression suite for dynamic Bloom filter
    • Added fail tests for dynamic Bloom filter intersection and union
  • API Changes
    • Added operator== and operator != to basic Bloom filter.
    • Implemented all methods for dynamic Bloom filter.

Version 0.4.0 is expected to arrive by July 4th with the introduction of a solid interface for the counting Bloom filter as well as up to date documentation.

v0.2.0 (June 24th)

  • Test Changes
    • Added tests for new constructors
    • Added tests for range insert()
    • Added tests for empty()
    • Added tests for swap()

v0.1.2 (June 23rd)

  • API Changes
    • Added constructor(begin, end)
    • Added constructor(initializer list)
    • Added global and member swap()
    • Added insert(begin, end)

v0.1.1 (June 21st)

  • API Changes
    • contains() -> probably_contains()
    • size() -> bit_capacity()
    • Added empty()

v0.1.0 (early June)

  • Initial release.
  • Features:
    • Basic Bloom filter class
    • One hasher object implemented based on Boost.Functional hash
    • Various examples: basic, advanced, url, string, custom_hasher
    • A test-suite that covers all basic functionality.
