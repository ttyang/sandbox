fuse_property_writer

Synopsis 

namespace boost {

    template <typename Filter, typename PropertyMap, typename OutputIterator>
    ::boost::fusion::pair<
        Filter
      , property_writer<PropertyMap,OutputIterator,Filter>
    >
        fuse_property_writer(PropertyMap property_map, OutputIterator out_itr);

}  // namespace boost

Description

Requires the Filter type to be explicitly passed in as a template parameter. Creates and returns a boost::fusion::pair that matches the filter type with a property_writer object that will be invoked when the algorithm reaches the appropriate event point.

Where defined 

#include <boost/graph/fuse_property_writer.hpp>
