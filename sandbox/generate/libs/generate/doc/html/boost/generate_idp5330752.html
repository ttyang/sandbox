Function template generate

boost::generate — Generates values for a range.

Synopsis

// In header: <boost/generate/generate.hpp>


template<typename Iterator, typename Generator> 
  void generate(Iterator first, Iterator last, Generator generator);

Description

This function dispatches to a range generate version iff the Generator used is a Range Generator for the Iterator employed. Otherwise the std::generate version is used.

Parameters:

first

Beginning of the range to be generated

generator

An instance of a Generator type

last

End of the range to be generated

Template Parameters:

Generator

A Generator type (function object)

Iterator

An Iterator type
