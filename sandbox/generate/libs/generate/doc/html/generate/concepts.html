Constant Stride Iterator

Description

A Constant Stride Iterator is an iterator whose values are guaranteed to be aligned with a constant stride in memory.

Refinement of

Random Access Iterator.

Notation

Expression

Description

X

A type that models the Constant Stride Iterator concept

i,j

Objects of type X

n

Integer constant: stride between the values the iterator is pointing to.

Invariants

Additional to the invariants defined by Random Access Iterator, the following must hold:

Name

Semantics

Constant stride access

if j == ++X(i), then (&(*j) - &(*i)) == n.

Models
  • T* (with stride 1)
  • std::vector<T>::iterator (with stride 1)

Unit Stride Iterator

Description

A Unit Stride Iterator is an iterator whose values are guaranteed to be aligned contiguously in memory.

Refinement of

Constant Stride Iterator.

Notation

Expression

Description

X

A type that models the Unit Stride Iterator concept

i,j

Objects of type X

Invariants

Additional to the invariants defined by Constant Stride Iterator, the following must hold:

Name

Semantics

Unit stride access

if j==++X(i), then (&(*j) - &(*i)) == 1.

Models
  • T*
  • std::vector<T>::iterator
